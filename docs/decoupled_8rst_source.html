3 Decoupled Front-end
4 ===================
5 
6 DKAN2 uses a `React front-end <https://github.com/GetDKAN/data-catalog-frontend>`_ built with `GastbyJS <https://www.gatsbyjs.org/>`_.
7 
8 The `frontend components <https://github.com/GetDKAN/data-catalog-components>`_ can be viewed here: https://GetDKAN.github.io/data-catalog-components
9 
10 Using the App
11 -------------
12 
13 DKAN comes with an integration module that allows the React App driving the frontend to be embedded in Drupal.
14 
15 To get the integration working follow these steps:
16 
17 - Place the source for the `data-catalog-frontend <https://github.com/GetDKAN/data-catalog-frontend>`_ inside of your ``docroot`` directory.
18 - Configure the ``.env.development`` and ``.env.production`` files to point to the backend urls.
19 - Install the dependencies with `npm <https://www.npmjs.com/>`_: ``cd data-catalog-frontend`` and ``npm install``
20 - Build: ``npm run build``
21 - Enable the integration module ``drush en dkan_frontend``
22 - Change the sites configuration to point the homepage (``/``) to ``/home``
